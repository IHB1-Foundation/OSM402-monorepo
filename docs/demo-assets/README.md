# 데모 에셋(복붙용)

실제 GitHub 데모에서 “보기 좋은” 흐름을 만들기 위해 필요한 복붙 템플릿 모음입니다.

- 이슈 예시: `docs/demo-assets/ISSUE_EXAMPLE.md`
- PR 예시(기본): `docs/demo-assets/PR_EXAMPLE.md`
- PR 예시(PASS): `docs/demo-assets/PR_PASS_EXAMPLE.md`
- PR 예시(HOLD): `docs/demo-assets/PR_HOLD_EXAMPLE.md`
- 데모용 코드베이스: `examples/bite-demo-repo/`
